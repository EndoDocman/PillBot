# PillBot: Geriatric-Friendly Personal Tablet Edition

A privacy-first, local-only medication tracker designed for tablets. PillBot uses a "Shadow Blockchain" for data integrity and provides high-contrast interfaces for elderly users.

## 🚀 Key Features
- **Geriatric UI:** Color-coded (Morning/Afternoon/Night) and extra-large targets.
- **Local-First:** Data stays in the browser's IndexedDB. No cloud required.
- **Shadow Blockchain:** Every entry is cryptographically linked (SHA-256) to ensure an immutable record.
- **Actionable Notifications:** Mark doses as "Taken" directly from the tablet's lock screen.
- **Emergency QR:** Read-only medical ID for first responders.

## 🛠 Setup
1. Open `index.html` on your tablet browser.
2. "Add to Home Screen" to install as a standalone PWA.
